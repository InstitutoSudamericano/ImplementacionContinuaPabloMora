# TindPablo API

API REST de una app de citas construida con NestJS + Prisma + PostgreSQL.

## Instalacion

```bash
npm install
```

## Base de datos

```bash
npx prisma db push
npx prisma db seed
```

## Ejecutar

```bash
npm start
```

- API: http://localhost:3000
- Swagger UI: http://localhost:3000/api
